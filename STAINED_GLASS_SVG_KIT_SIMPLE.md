# Stained Glass SVG Quick Guide (Canva)

Use this as the simple production checklist.

## 1) Colors To Use

Use only these colors so everything matches:

- Warm amber glass: `#D8C38D`
- Soft teal glass: `#9EB5B4`
- Burnt orange glass: `#B36A3A`
- Pale gold highlight: `#EAD9A1`
- Came metal (main): `#5C4A3F`
- Came dark edge: `#43352D`
- Came light highlight: `#F4E8CE`

Opacity guidance:
- Glass panes: about `30% - 60%` opacity
- Came lines: mostly solid (`70% - 95%` opacity)

## 2) Exact Assets We Need

Create these files in SVG format:

### Viewport corners
- `corner-top-left.svg`
- `corner-top-right.svg`
- `corner-bottom-left.svg`
- `corner-bottom-right.svg`

### Card corners
- `card-corner-a.svg`
- `card-corner-b.svg`

### Cames
- `came-horizontal.svg`
- `came-vertical.svg`

### Optional texture
- `glass-grain-overlay.svg`

## 3) Critical Rules (Most Important)

- Background must be **fully transparent** (no white rectangle behind art).
- Export as **SVG**.
- Keep shapes simple and clean (avoid too many tiny details).
- Use the same color palette for every asset.
- Keep line style consistent for all cames.
- Make sure each corner asset is oriented correctly for its named corner.
- Do not include text in the SVG files.
- Keep file names exactly as listed above.

## 4) Suggested Sizes (Easy Defaults)

- Corner assets: `360 x 360`
- Card corner assets: `128 x 128`
- Horizontal came: `600 x 10`
- Vertical came: `10 x 600`
- Grain tile: `220 x 220`

## 5) Quick Export Check Before Sending

For each SVG:
- Open it in a browser.
- Confirm transparency works.
- Confirm it is the right corner/orientation.
- Confirm colors match the list above.
- Confirm filename matches exactly.
